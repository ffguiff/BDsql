# BDsql
estudo e formulas usadas para manipulação de dados sql. 
esse repositorio é para armazenar e demostrar algumas aptidões com sql
com DDL para criação e estruturação de tabelas e do banco de dados;
DML para manipulçao de dados inserido, alterando e deletando, informações, colunas  e tabelas;
DQL para analise de dados fintrando informações e gerando novas metricas apartir de requisições exigidas para diversos tipos de analises.
