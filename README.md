# BDsql
estudo e formulas usadas para manipulação de dados sql. 

Este repositório foi desenvolvido para demonstrar competências técnicas em SQL, com foco na estruturação, manipulação e análise de dados para suporte à tomada de decisão.

Aptidões Demonstradas:

* **DDL (Data Definition Language):** Criação e estruturação lógica do banco de dados e tabelas, garantindo a integridade dos tipos de dados e relacionamentos.
* **DML (Data Manipulation Language):** Gerenciamento do ciclo de vida dos dados, incluindo inserção, atualização e deleção de registros para manutenção da base.
* **DQL (Data Query Language):** Foco principal em **Análise de Dados**, utilizando filtros complexos e funções de agregação para transformar dados brutos em métricas e insights relevantes.
